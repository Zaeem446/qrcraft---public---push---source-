# QRFY Research — Missing Types & Features

## Missing QR Code Types (QRFY has, we don't)
- **Landing Page** — Full drag-and-drop page builder with blocks (text, image, video, button, social icons)
- **Product** — Product showcase with image gallery, price, description, buy button
- **Feedback / Survey** — Star rating + comment form, results dashboard
- **Playlist** — Multi-platform music links (Spotify, Apple Music, YouTube Music, etc.)
- **Phone Call** — Direct dial link with name/number display
- **Calendar Event** — .ics download with date/time, location, description, RSVP

## Missing Features on Existing Types

| Type | Missing Features |
|------|-----------------|
| Menu | Category images, item prices, item photos, dietary icons |
| Business | Business hours, embedded map, multiple locations |
| Coupon | Expiry date/countdown, terms & conditions, barcode |
| Event | RSVP form, calendar add (.ics), countdown timer, map embed |
| vCard | Social media links, company logo, multiple phone/email |
| Links | Button style options (rounded/square/outline), thumbnails per link |
| Review | Multi-platform review links (Google, Yelp, TripAdvisor) |

## QRFY Brand Templates System
QRFY offers reusable "brand templates" for QR code visual design:
- Save QR dot style, colors, logo, frame as a reusable template
- Apply to any new QR code with one click
- Useful for businesses maintaining brand consistency

## Priority Suggestions
1. **High:** Landing Page builder (most requested type), Product page, Feedback form
2. **Medium:** Playlist, Calendar Event, enhanced Menu with prices/images
3. **Low:** Phone Call (simple), brand templates system
